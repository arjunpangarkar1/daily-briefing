# Daily Briefing

A personal daily news dashboard that surfaces financial, geopolitical, and economic headlines each morning in a clean, dark-themed interface.

## Features

- **Live market tickers** — S&P 500, NASDAQ, Gold, Oil (WTI), and USD/EUR updated daily via Alpha Vantage
- **Finance** — Top market and equities news via Marketaux
- **Geopolitical** — World news and foreign policy coverage via The Guardian
- **Economy** — Macroeconomic headlines via The Guardian
- **Central Banks & IMF** — Monetary policy and interest rate news via Marketaux

## Built With

- HTML, CSS, JavaScript (no frameworks)
- [Marketaux API](https://marketaux.com) — financial news
- [Guardian API](https://open-platform.theguardian.com) — world and economic news
- [Alpha Vantage API](https://alphavantage.co) — market data
- Hosted on GitHub Pages
